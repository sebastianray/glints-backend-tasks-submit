Berikut adalah fungsi-fungsi command dalam git yang telah kami pelajari di hari ke-3 ini :
- git checkout -b <branch_name>, 
Perintah ini digunakan untuk membuat branch baru sekaligus berpindah dari master ke branch tersebut.
- git pull origin <branch_name>, 
Perintah ini berfungsi untuk menarik branch remote, lalu menggabungkannya dengan local branch yang sedang dijalankan.
